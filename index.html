Muse (final name pending) is my attempt to combine the various tools I use for solo tabletop roleplaying and for when I GM.

The interface I've kept minimal to make it easy to use and maintain.

The journal log (below) is not stored in the app. It's designed to be copied out into whatever word processing software you use, on desktop or mobile. It doesn't provide rich text formatting as the coding around that gets a lot more complicated, especially for consistent experience between desktop and mobile devices.
