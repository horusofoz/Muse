Party tier refers to the level of the party in play.

This is used to set level appropriate difficulty classes and consequences for skill challenges and saving throws.

Wish is a tool for tabletop role-playing games.

The basic appearance ensures a distraction free experience for the GM or solo player generating content on the fly. Little to no graphics beyond buttons icons are included.

Navigation

To access a feature, first select from the menu items above to access a feature set. Then select the specific feature on the left sidebar. For example, to access the Oracle's Question feature, select Oracle on the top menu, then Question on the left sidebar.

Learn about a feature

To learn more about a feature, select About (already here) then select the feature set it belongs to in the left sidebar. For example to learn more about the Oracle's Question feature, select About from the top menu then Oracle on the left sidebar.

Saving Your Journal

Content generated goes into the journal for copying to your preferred text editor. The journal is not saved in the app. Copy it out at the end or during your session to avoid losing progress.

A log of generated content.

Write
Directly write entries to the journal. Scene setups, dialogue, etc.

  • Append adds the written journal entry to the log.
  • Erase clears the in progress journal entry.

Read
Maximizes the journal space for reading.

Undo
Removes the last entry from the journal. This applies generated content as well as written journal entries.

Copy
Copies the entirety of the log to the clipboard for saving into your preferred text editor.

Clear
Wipes all entries from the journal.
